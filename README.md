# ocs-flashcards

Flashcards have always been my favorite way to study.
I have created this web application in order to study for OCS while spending time on the computer.


Feel free to add your own flashcard decks if you like.

I have added some decks I feel are important,
but as they say two minds are better than one.

Therefore, if you have any ideas, please create a new deck instead of adding on to another persons deck.
This creates an environment in which we can learn from each other.

It is pretty self explanatory.  

Click (+ NEW DECK) to create a new deck and you will be prompted to name it.
Feel free to include your name in the format of (OC XXXXX), so we as a team can know who has contributed.

Click (MANAGE) to enter the create a new card window.
Click (ADD) when you have filled in the question and the answer.
Rinse and repeaat to create as many as you like. 

(TEST) allows you to test yourself, but be warned, the answer has to match exactly to the registered answer.

(STUDY) allows you to flip through the cards one at a time with the option of shuffling them.

Please DM me on WhatsApp if you catch any bugs in this app, so I can fix it, and if you have any ideas to improve upon it as well.  

I hope this helps each memeber of the team become a satisfactory Officer Candidate in order for each of us to graduate.
