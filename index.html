OCS Flashcards

Flashcards have always been my favorite way to study. I created this web application in order to study for OCS while spending time on the computer.

Feel free to add your own flashcard decks. I have added some decks I feel are important, but as they say — two minds are better than one.

If you have any ideas, please create a new deck instead of adding on to another person's deck. This creates an environment in which we can learn from each other.

How to use

+ New Deck — Create a deck. Include your name as (OC XXXXX) so the team knows who contributed. Click the pencil icon to rename the deck.

Manage — Add cards. Fill in a question and answer, then click Add. Repeat as many times as you like. Click the pencil icon to edit current flashcards.

Test — Test yourself. Note: your answer must match exactly.

Study — Flip through cards one at a time. Shuffle anytime.

Please DM me on WhatsApp if you catch any bugs or have ideas to improve the app.

I hope this helps each member of the team become a satisfactory Officer Candidate so that each of us may graduate.
